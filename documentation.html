Телеграм-бот @phystech_meet_bot на Python.

Содержание репозитория

ФайлНазначение
main.pyГлавный файл с основной логикой.
create_bd.pyМодуль для создания базы данных.
bd.pyМодуль для работы с базой данных.
bot_data.pklФайл, хранящий состояния пользователей.
requirements.txtСписок зависимостей.

Сетевая состовляющая

  • Бот размещен на хостинге U1 HOST.
  • Вся сетевая логика проекта сосредоточена в модуле main.py, который работает с Telegram API: он получает входящие обновления от серверов Telegram и отправляет пользователям сообщения.

Общественное признание

  • В боте зарегистрировалось более 150 человек и было совершено более 3000 взаимодействий между пользователями.
